## Built With

*   Bootstrap / JQuery
*   Laravel

## Getting started

### Production

*   cp .env.example .env
*   php artisan key:generate
*   Check default .env
*   Check ../storage
*   Check server config
*   composer install
*   npm install
*   npm run prod

### Develop

*   cp .env.example .env
*   php artisan key:generate
*   Check default .env
*   Check ../storage permissions
*   Check server config
*   composer install
*   npm install
*   npm run watch-poll

## Console commands

*   php artisan make:sitemap - sitemap generator
